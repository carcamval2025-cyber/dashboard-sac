# Pizarra de Calidad del Servicio — Sivareña Pizzas (SERVQUAL)

Dashboard interactivo de un estudio de calidad del servicio con el modelo **SERVQUAL**
(Sivareña Pizzas, sucursal Los Héroes, San Salvador · ciclo 01/2026).

- **Ver en línea:** https://carcamval2025-cyber.github.io/dashboard-sac/
- Archivo único autocontenido (`index.html`): incluye librería de gráficos, datos, fuentes y logo.
- Funciona 100% offline y sin servidor.
- **Datos anonimizados**: solo `id` y `género`. Nombre, teléfono y correo de los encuestados fueron excluidos por privacidad.

Universidad Doctor Andrés Bello · Servicio y Atención al Cliente.
